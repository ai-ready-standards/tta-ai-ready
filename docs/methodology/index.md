# 매핑 자료실

본 사업의 AI 레디 패키지가 어떻게 만들어졌는지를 보여주는 모든 작업물입니다. **TTA 표준 → 글로벌 어휘 매핑 → AP 패키지** 4단계 프로세스의 산출물입니다.

## 4단계 워크플로우

```
Phase A: 표준 본문 추출        →  tta-standards/0976/
Phase A.5: 누락 어휘 보강      →  inventory/ 확장 (319 → 478행)
Phase B: 매핑 매트릭스 작성    →  mappings/
Phase C: AP 패키지 작성         →  standards/P-XX/
Phase D-1: 자기 검증            →  reports/phase_d1_verification.md
```

## 4가지 자료

### 1. [인벤토리 (478행)](inventory.md)

11종 국제 어휘 + TTA 표준 요소를 통합한 마스터 인벤토리. 각 항목에 `concept_tag`(주제 분류)과 `equivalent_terms`(동등 어휘) 포함.

### 2. [매핑 매트릭스 (93행 × 7컬럼)](mappings.md)

TTAK.KO-10.0976의 93개 요소 각각이 7개 구성요소(c1~c7)에 어떻게 매핑되는지를 매트릭스로 정리. 우선순위·신뢰도·결정 근거 포함.

### 3. [보고서 (Phase A~D-1)](reports.md)

10건의 단계별 보고서. 작업 과정·통계·결정 사항·발견된 버그까지 모두 추적.

### 4. AP 패키지

각 표준의 [`standards/P-XX-domain/`](../standards/index.md) 디렉토리에서 6개 디렉토리 구조로 발행됩니다.
