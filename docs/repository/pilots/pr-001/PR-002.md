## 변경 상세

### 변경 ① — 어휘·표준 표 통합

기존 2개 하위표 → 단일 통합표 + 발행 기관 컬럼.

**기존**:
```
### W3C 표준 어휘 (RDF 발행)        ← 9개 행
### 메타데이터 표준 (외부 기관)      ← 8개 행
```

**변경 후**:
```
### 외부 어휘·표준 (통합)            ← 18개 행 (기존 17 + schema.org 1)
| 약어 | 풀이 | 발행 기관 | 본 사업 사용처 |
```

### 변경 ② — schema.org 행 추가

| 약어 | 발행 기관 | 본 사업 사용처 |
| --- | --- | --- |
| schema.org | schema.org Community Group | 외부 정합성 매핑 (`aird-datatype-to-schemaorg.ttl`, PR-001 도입) |

### 변경 ③ — 표준화 기관 표 보강

기존 6개 → 11개 (추가 5개):
- schema.org Community Group
- DCMI (Dublin Core Metadata Initiative)
- DDI Alliance
- SDMX Consortium
- EBI / EBISPOT
