## 빠른 시작

### 1. 시작 템플릿 다운로드

가장 단순한 데이터셋 메타데이터 템플릿입니다. 6개 필수(M) 필드만 포함되어 있습니다.

[:material-download: dataset-minimal.jsonld](https://raw.githubusercontent.com/ai-ready-standards/tta-ai-ready/main/standards/P-01-research-data/examples/valid/dataset-minimal.jsonld){ .md-button .md-button--primary }

```json
{
  "@context": "https://raw.githubusercontent.com/ai-ready-standards/tta-ai-ready/main/standards/P-01-research-data/schema/context.jsonld",
  "id": "https://doi.org/10.7488/ds/example-001",
  "type": "Dataset",

  "identifier": "https://doi.org/10.7488/ds/example-001",
  "identifierType": "DOI",
  "title": {"en": "My Research Dataset"},
  "creator": {"type": "Person", "name": "Your Name"},
  "publisher": {"type": "Organization", "name": "Your Institution"},
  "publicationYear": "2026"
}
```

### 2. 검증

```bash
pip install -e tools/validator/
tta-validator your-dataset.jsonld
```

→ [상세 검증 가이드](../validate.md)

### 3. PyTorch · HuggingFace에서 사용 (Phase B 예정)

Croissant 1.0 호환 어댑터가 추가되면 다음과 같이 한 줄 로딩 가능해집니다.

```python
import mlcroissant as mlc
ds = mlc.Dataset(jsonld="path/to/your-dataset.jsonld")
records = ds.records("default")
```
