    TTA AI 레디 표준¶

    "사람이 읽는 표준"에서 "AI가 직접 임포트하는 표준"으로

    한국정보통신기술협회(TTA)의 AI 레디 표준 생태계 조성 사업의 공개 카탈로그입니다. 5종 시범 표준이 JSON-LD 스키마 · SHACL 검증 규칙 · 참조 구현 코드 · 예시 데이터셋 패키지로 제공됩니다.

    본 사이트의 두 종류 콘텐츠

    좌측 네비게이션은 두 그룹으로 명확히 분리되어 있습니다.

    그룹 정체 누구를 위한 것
    표준 (Standards) 5종 시범 표준의 AI Ready Data 형식 패키지 — 본 사업의 결과물 표준 사용자, 시스템 구현자, 데이터 발행자
    사업 방법론 (Project Methodology) 5종 표준을 만들고 운영하기 위한 프레임워크·매뉴얼·도구·파일럿 — 본 사업의 운영 자료 표준 개발자, PG606 위원, 외부 기여자, 평가위원

    공통 참조: 용어집·약어 는 두 그룹 모두에서 인용됩니다 (페이지 상단에 영역 분류 안내).

    5종 표준 둘러보기 새 표준 제안하기 직접 검증해보기 용어집 GitHub

    본 사업이 약속하는 것¶

    • 즉시 import

      import mlcroissant 한 줄로 표준 로딩. 글로벌 ML 프레임워크(PyTorch · TensorFlow)와 직접 연동

    • 글로벌 검색 가능

      Google Dataset Search · HuggingFace · re3data 등 700,000+ 데이터셋 생태계와 호환

    • 자동 검증

      SHACL 기반 적합성 검증을 CI 파이프라인에서 자동 실행. 사람의 수동 검토 불필요

    • 개방형

      Apache License 2.0. 산학연 자유 활용 보장
