# 4개 목적별 프로파일

각 프로파일은 동일한 7개 스키마 구성요소를 특정 활용 목적에 맞게 구성한 적용 가이드입니다. 프레임워크 정의서(B-3 산출물)에서 정식으로 정의됩니다.

| 프로파일 | 적용 목적 | 핵심 어휘 |
| --- | --- | --- |
| [`ml/`](./ml/) | ML 학습 파이프라인 | Croissant 1.0, DCAT v3 |
| [`rag/`](./rag/) | RAG (검색 증강 생성) | DCAT v3, schema.org |
| [`kg/`](./kg/) | 지식 그래프 구축 | OWL, SKOS, PROV-O |
| [`statistics/`](./statistics/) | 통계 분석 | DDI, SDMX, ISO 5259 |
